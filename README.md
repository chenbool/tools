# 常用工具函数

PHP 常用工具函数集合，包含快递查询和汉字验证功能。

## 功能特性

- 快递查询（支持多家快递公司）
- 汉字验证（判断是否包含中文）

## 文件说明

| 文件 | 功能 |
|------|------|
| `express.php` | 快递查询 |
| `hanzi.php` | 汉字验证 |

## 快递查询

### 函数说明

```php
get_express($sn)
```

### 参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `$sn` | string | 快递单号 |

### 返回值

返回物流信息数组，包含以下字段：

| 字段 | 说明 |
|------|------|
| `comCode` | 快递公司代码 |
| `message` | 物流状态 |
| `time` | 时间节点 |
| `context` | 物流详情 |

### 使用示例

```php
include 'express.php';

$res = get_express('3924730222455');
print_r($res);
```

## 汉字验证

### 函数说明

```php
is_Chinese($str)
```

### 参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `$str` | string | 待验证字符串 |

### 返回值

- 全是中文
- 含有中文
- 没有包含中文

### 使用示例

```php
include 'hanzi.php';

$str = '开放所有函数,超时时间';
is_Chinese($str);  // 输出：含有中文
```

## 依赖

- PHP 5.6+
- cURL 扩展（用于快递查询）
